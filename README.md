# Sistema Cognitivo de Inventario con MCP y SQLite

Este proyecto corresponde al desarrollo de un servidor MCP en Python para la gestión de un inventario, usando SQLite como base de datos local.

## Objetivo

Diseñar e implementar un servidor MCP que permita realizar operaciones CRUD y consultas analíticas sobre una base de datos de inventario, con el fin de comprender cómo un sistema cognitivo puede interactuar con herramientas externas.

## Tecnologías utilizadas

- Python
- SQLite
- FastMCP
- Visual Studio Code
- GitHub

## Funcionalidades

- Crear productos
- Consultar productos por ID
- Actualizar cantidad de productos
- Eliminar productos
- Listar productos registrados
- Calcular el valor total del inventario
- Consultar productos agotados
- Identificar el producto más costoso
- Consultar estadísticas generales del inventario
